\# Agile Task Board



Sistema de Gestión de Tareas para Equipos Ágiles.



\## Descripción

Esta es una aplicación web simple para gestionar tareas en un tablero Kanban, usando HTML/CSS/JS y LocalStorage.



\## Estructura

\- `docs/`: Documentación (backlog, arquitectura).

\- `src/`: Código fuente.

&nbsp; - `components/`: Componentes reutilizables.

&nbsp; - `styles/`: Estilos CSS.
